Escribe un programa que ponga en práctica la siguiente lógica (3.5 ptos): a) Mostrar el siguiente menú y el usuario deberá elegir una de las opciones b) En función de la opción que elija el usuario se le pedirán los datos necesarios en cada caso. c) El programa calculará el resultado y lo mostrará en la consola. d) La lógica anterior se repetirá indefinidamente hasta que el usuario pulse la opción 4 para salir que mostrará un mensaje por consola, Fin del ejercicio. e) Si la opción no está entre el rango 1-4 se mostrará un error mediante la función alert() y se volverá a pedir otra opción
